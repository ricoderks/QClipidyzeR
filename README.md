# QClipidyzeR

<!-- badges: start -->
<!-- badges: end -->

The goal of `QClipidyzeR` is to have a quick look at the quality of a
Lipidyzer study.

## Installation

You can install the development version of `QClipidyzeR` like so:

``` r
remotes::install_github("ricoderks/QClipidyzeR")
```

## Run

Run the app with:

``` r
QClipidyzeR::run_app()
```
